# Math Notes Formatter

Този skill помага при форматирането на математически записки в Markdown формат.

## Правила за форматиране

### 1. Математически формули

**Важно:** Всички математически формули трябва да бъдат обградени с `$` (за inline) или `$$` (за block формули).

**Правилно:**
- Inline: `$\alpha \in \Sigma^*$`
- Block:
  ```
  $$
  L / \alpha = \{\beta \in \Sigma^* \ | \ \alpha\beta \in L\}
  $$
  ```

**Грешно:**
- α ∈ Σ⋆ (без долари)
- L/α = {β | αβ ∈ L} (без долари)

### 2. Специални символи

- **Празна дума:** `$\varepsilon$` вместо ε или epsilon
- **Звезда на Клини:** `$\Sigma^*$` вместо Σ⋆ или Σ*
- **Празно множество:** `$\emptyset$` вместо ∅
- **Принадлежност:** `$\in$` вместо ∈
- **Подмножество:** `$\subseteq$` вместо ⊆
- **Обединение:** `$\cup$` вместо ∪
- **Сечение:** `$\cap$` вместо ∩
- **Разлика:** `$\setminus$` вместо \
- **Допълнение:** `$\overline{L}$` вместо L̄
- **Точки:** `$\ldots$` вместо ... (за последователности)
- **Не равно:** `$\neq$` вместо ≠
- **Импликация:** `$\implies$` или `$\Rightarrow$` вместо ⟹

### 3. Структура на документа

```markdown
# Главна тема

## 1. Първа секция

### 1.1. Подсекция

- Буллети с информация
- **Важни термини** трябва да са в bold
- Дефиниции трябва да са ясно отделени

**Примери:**
- Пример 1
- Пример 2

**Свойства:**
- Свойство 1
- Свойство 2
```

### 4. Дефиниции

Формат за дефиниции:
```markdown
**Име на понятието:** Кратко обяснение на понятието.

**Формално:** Математическата дефиниция с формули.
$$
\text{Формула}
$$

**Интуитивно:** Обяснение с думи какво означава.

**Примери:**
- Конкретни примери
```

### 5. Доказателства

Формат за доказателства:
```markdown
**Твърдение:** Формулировка на твърдението.

**Доказателство:**

**База на индукцията:**
- Случай 1

**Индуктивна стъпка:**
- Допускане
- Разглеждаме случай X
- Заключение

**□** (край на доказателство)
```

### 6. Често използвани LaTeX команди

| Значение | LaTeX команда | Резултат |
|----------|--------------|----------|
| Буква алфа | `$\alpha$` | $\alpha$ |
| Буква бета | `$\beta$` | $\beta$ |
| Буква гама | `$\gamma$` | $\gamma$ |
| Буква делта | `$\delta$` | $\delta$ |
| Сигма (азбука) | `$\Sigma$` | $\Sigma$ |
| Сигма звезда | `$\Sigma^*$` | $\Sigma^*$ |
| Епсилон | `$\varepsilon$` | $\varepsilon$ |
| За всяко | `$\forall$` | $\forall$ |
| Съществува | `$\exists$` | $\exists$ |
| Логическо И | `$\land$` | $\land$ |
| Логическо ИЛИ | `$\lor$` | $\lor$ |
| Отрицание | `$\neg$` | $\neg$ |
| Обединение | `$\bigcup_{i=0}^{n}$` | $\bigcup_{i=0}^{n}$ |
| Сечение | `$\bigcap_{i=0}^{n}$` | $\bigcap_{i=0}^{n}$ |
| Множество | `$\{a, b, c\}$` | $\{a, b, c\}$ |
| Функция | `$f: A \to B$` | $f: A \to B$ |
| Стрелка | `$\Rightarrow$` | $\Rightarrow$ |
| Двойна стрелка | `$\iff$` | $\iff$ |

### 7. Често срещани грешки

❌ **Грешно:**
- Използване на Unicode символи директно без долари
- Смесване на обикновен текст и математика в един израз
- Липса на разстояния в математически изрази

✅ **Правилно:**
- Всички математически символи в `$...$`
- Разделяне на текст и математика
- Използване на `\ ` за интервали в математика: `$\alpha \ | \ \beta$`

### 8. Примери за типични конструкции

**Множество с условие:**
```latex
$L = \{\alpha \in \Sigma^* \ | \ \text{условие за } \alpha\}$
```

**Функция:**
```latex
$f: A \times B \to C$
$f(x, y) = \text{израз}$
```

**Индукция:**
```latex
**База:** При $n = 0$ имаме $P(0)$.

**Стъпка:** Нека $P(n)$ е вярно. Ще докажем $P(n+1)$.
```

- [ ] Дефинициите са ясно отделени и форматирани
